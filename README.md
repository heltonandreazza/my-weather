# my-weather

The app uses:

- AngularJS 1.5.x
- Bootstrap 3.3.x
- AdminLTE 2.3.x
- Ionicons 2.0.x
- Font Awesome 4.6.x

## Installation and usage

Install npm and run with start:
```bash
$ npm install webpack -g
$ npm install webpack-dev-server -g
$ npm install
$ npm start
```

Generating build:
```bash
$ npm run build or webpack
```
