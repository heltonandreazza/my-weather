# my-weather

The app uses:

- AngularJS 1.5.x
- Bootstrap 3.3.x
- AdminLTE 2.3.x
- Ionicons 2.0.x
- Font Awesome 4.6.x

## Installation and usage

Install yarn and run with start:
```bash
$ yarn install
$ yarn start
```

Generating build:
```bash
$ yarn run build
```
